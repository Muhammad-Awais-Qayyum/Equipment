# Equipment Management System

School sports equipment borrowing system with PWA support for offline functionality.

## Features

- Equipment inventory management
- Student borrowing system
- Admin and Sports Captain portals
- PWA (Progressive Web App) support
- Offline functionality with Dexie/IndexedDB
- Installable on desktop and mobile devices

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## PWA Features

- Installable as a native app
- Works offline
- Service worker caching
- Local data storage with Dexie
